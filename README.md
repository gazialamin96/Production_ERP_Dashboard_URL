**Production ERP Dashboard & Digital Operations System**

**Project Description:**
Developed and implemented a comprehensive ERP dashboard system designed for production monitoring and executive-level decision-making (Chairman, CEO, COO). The system was originally developed by Logic Software, and I handled full implementation, customization, and business logic calculations based on company requirements.

In addition, I built a responsive web portal using HTML5, CSS3, and JavaScript to ensure seamless access across desktop, tablet, and mobile devices. The system provides real-time insights into production performance, efficiency, and operational KPIs.

Beyond ERP, I contributed to digital transformation by working on a custom MDM (Mobile Device Management) system to remotely control and monitor ~70 smart TVs from the head office. I also supported the development and deployment of an Android-based Smart Tracking application for End Line Quality Control (QC) in production.
